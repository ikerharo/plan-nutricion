## Prep semanal (domingo, ~1.5 hrs)

### Proteinas (30 min):
- [ ] Hervir 750g pechuga. Desmenuzar. 3 contenedores de 250g → L/Ma/Mi
- [ ] Hervir 6 huevos duros → snacks

### Caldo (15 min activos + 2-4 hrs pasivas):
- [ ] Huesos + cebolla + ajo + apio + zanahoria + sal
- [ ] Fuego bajo 2-4 hrs. Colar. 5-7 porciones → cenas

### Ensalada "Jugo 2" (10 min):
- [ ] Picar jitomate, apio, perejil, cebolla para 3 dias. Sin aliño
- [ ] Repetir miercoles

### Granos (15 min):
- [ ] Arroz para 3 dias (3 tazas secas)
- [ ] O quinoa

### Ingredientes de jugos (15 min):
- [ ] Lavar todo
- [ ] Separar por porcion diaria en bolsas
- [ ] Curcuma, jengibre, sabila: no pre-cortar
