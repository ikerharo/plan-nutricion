# Dieta de Eliminacion y Reintroduccion

## Fase 1: Eliminacion (ahora - Jun 16)

### No comer:
| Categoria | Incluye | Por que |
|---|---|---|
| Lacteos | Leche, queso, yogurt, crema, helado | Sospechoso en DA. Estimula acido gastrico |
| Gluten | Trigo, cebada, centeno, pan, pasta, tortilla harina | Sospechoso en DA. Puede inflamar mucosa gastrica |
| Azucar refinada | Dulces, refrescos, postres industriales | Proinflamatorio. Alimenta hongos |
| Ultraprocesados | Empaquetados, embutidos, snacks industriales | Emulsificantes erosionan moco intestinal |
| Frituras | Todo lo frito | Estres oxidativo, retrasa vaciamiento gastrico |
| NSAIDs | Ibuprofeno, naproxeno, aspirina | Causa #1 de gastritis no-H.pylori |
| Atun de lata | Enlatado | Mercurio. Preferir pescado fresco |
| Alcohol | Todo tipo | Irritante gastrico directo |

### Si comer:
| Categoria | Ejemplos |
|---|---|
| Proteinas | Pollo, salmon, res, cerdo, camarones, atun fresco, huevo, pavo |
| Carbohidratos | Arroz, quinoa, avena, papa, camote, platano, tortilla de maiz, pasta sin gluten |
| Grasas | Aguacate, aceite oliva, aceite coco, nueces, almendras |
| Verduras | Todas, cocidas de preferencia (gastritis) |
| Frutas | Todas, especialmente berries |
| Caldos | Pollo, hueso, verduras |

### Excepciones:
- **Isopure 0 Carbs:** Es whey isolate (sin caseina). OK durante eliminacion
- **Tortilla de maiz:** Sin gluten. OK
- **Creatina:** OK. No alimenta parasitos
- **Soya:** OK en cantidades normales (salsa soya para bowls)

## Fase 2: Reintroduccion (Jun 16+)

### Protocolo:
1. Reintroducir UN grupo a la vez
2. Comer ese grupo 2-3 dias seguidos
3. Observar 3-5 dias: gastritis, digestion, piel, energia
4. Si no hay reaccion → ese grupo esta OK, mantener
5. Si hay reaccion → identificaste un trigger. Eliminar de nuevo

### Orden sugerido:
| Semana | Reintroducir | Que probar | Observar |
|---|---|---|---|
| Jun 16-20 | Lacteos suaves | Yogurt natural, queso fresco | Gastritis, heces, piel |
| Jun 21-25 | Lacteos fuertes | Leche, queso manchego, crema | Mismo |
| Jun 26-30 | Gluten | Pan, pasta de trigo, tortilla harina | Mismo + energia, hinchazon |
| Jul 1+ | Todo junto | Pizza, hamburguesa con pan | Confirmacion |

### Si planeas prueba de playa:
- Ir a la playa DURANTE la reintroduccion, idealmente despues de cada grupo
- Si la piel reacciona diferente a lo normal → correlacionar con lo que reintrodujiste
- Si la piel reacciona igual que siempre → el trigger no es dietetico

## Fase 3: Largo plazo (80/20)

- **80% en rutina:** Cocina limpia, suplementos base (T1), evitar ultraprocesados
- **20% en social/viajes/finde:** Pizza, hamburguesa, queso, cerveza — sin culpa
- **Lo que SIEMPRE mantener:** Omega-3, vitamina D, creatina, magnesio, psyllium, entreno, sueno
- **Lo que puedes dejar cuando quieras:** T2 (gastritis), T4 (vitaminas base), jugos, bebidas de Ale
